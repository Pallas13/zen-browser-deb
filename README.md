# What is this repository? 

Convert Zen Browser of tar.gz for .deb package for Linux based in Debian how like (Ubuntu, Debian, Pop OS, Zorin OS) with icon for launcher.

file .desktop is of Flatpak version, copied of .local/share/flatpak/app/app.zen_browser.zen/current/active/files/share/applications/app.zen_browser.zen.desktop.

# Caution 

This not is a version official of Zen Browser, is just tar.xz version in deb package. Issues about .deb package is this repository and Issues for Zen Browser is repository official of Zen Browser.
